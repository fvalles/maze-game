Basic open modal component:

```jsx
<Modal
  height={200}
  isOpen={true}
  label="Some label"
  testId="some-test-id"
  width={200}>
  <H1>Lorem ipsum</H1>
</Modal>
```

Basic closed modal component:

```jsx
<Modal
  height={200}
  isOpen={false}
  label="Some label"
  testId="some-test-id"
  width={200}>
  <H1>Lorem ipsum</H1>
</Modal>
```
